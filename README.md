# Skull app

Программа для показа черепа по нажатию кнопки.

**Пример: **
<img width="1302" height="866" alt="image" src="https://github.com/user-attachments/assets/6e8111bc-e167-4a67-899f-0ede3cd0a83d" />

Имеется возможность настраивать бинды и сворачивать в трей.

Для запуска установите последний релиз и запустите exe файл.
Для самостоятельной сборки склонируйте репо, запустите pip install -r requirements.txt и py main.py.
